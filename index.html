計算公式

  • 面板暴擊率 = C / (A + H + P) × 100%
  • 面板治療量 = A + C + H + P
  • 攻爆奶進圖暴擊率 = ((C × 1.2) + 40) / (A + H + P) × 100%
  • 攻爆奶進圖治療量 = 面板治療量 × 1.096875
  • 純攻/攻穿奶進圖暴擊率 = (C × 0.989) / (A + H + P) × 100%
  • 純攻/攻穿奶進圖治療量 = 面板治療量 × 1.07625
