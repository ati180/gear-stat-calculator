計算公式

  • 先算面板基底：B = A + H + P
  • 面板暴擊率 = C / B × 100%
  • 面板治療量 = B + C
  • 攻爆奶進圖後：攻擊力變成 A × 1.125，暴擊變成 C × 1.2625
  • 純攻/攻穿奶進圖後：攻擊力變成 A × 1.2375，暴擊變成 C × 1.0675
  • 進圖暴擊率 = 進圖後暴擊 /（進圖後攻擊力 + H + P），最高不超過 100%
  • 進圖治療量 =（進圖後攻擊力 + H + P）×（1 + 進圖暴擊率）

符號對照

  • A = Attack（攻擊力）
  • C = Crit（暴擊）
  • H = Hit（命中）
  • P = Penetration（穿透）

計算公式

  • 面板暴擊率 = C ÷（A + P）
  • 面板穿透率 = P ÷ A
  • 傷害 = A ×（1 + 暴擊率）×（1 + 穿透率）×（1 + D%）
  • 暴擊率上限 100%，超出不計
  • 各職業天賦會依權重（Wa、Wp）修正暴擊率：天賦暴擊率 = C ÷（Wa × A + Wp × P）

符號對照

  • A = Attack（攻擊）
  • C = Crit（暴擊）
  • P = Penetration（穿透）
  • D = Damage Bonus（增傷 %）
  • Wa = 攻擊權重、Wp = 穿透權重（見下表）
